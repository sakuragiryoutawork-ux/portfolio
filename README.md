# ポートフォリオサイト

## 概要

WordPressを使用して制作したポートフォリオサイトです。

## 使用技術

・WordPress  
・PHP  
・HTML / CSS / JavaScript

## 機能

・カスタム投稿機能  
・お問い合わせフォーム

## 工夫した点

・キービジュアルにはposition: sticky;を使用し、背景画像がスクロールに応じて固定されるレイアウトを実装しました。
一方で、キービジュアルにあるテキストがスクロール後も背景上に残ってしまう問題があったため、JavaScriptを用いて特定の位置でテキストをフェードアウトさせる処理を追加しています。
シンプルな構成の中でも違和感のない動きになるよう調整し、見やすさとデザイン性の両立を意識しました。

・また、カスタム投稿とタクソノミーを活用し、カテゴリーごとに記事を整理・表示できるページを実装しました。
ユーザーが目的の情報にアクセスしやすいよう、構造設計を意識しています。

・お問い合わせ機能では、入力フォーム・内容確認ページ・送信完了ページをそれぞれ作成し、ユーザーが入力内容を確認できる構成にしました。
また、データの保存にはプラグインを活用し、送信された内容をデータベースに保存できるようにしています。

## 苦労した点

・動きを取り入れつつ、 全体のデザインバランスを崩さない点に苦労しました。
